js-chat by Oshuma

Simple Javascript chat type dealy.

Dependencies

HTML & Javascript (The examples use jQuery, but it's not a requirement).

Install

Include js-chat.js in your layout, then take a look at the examples.

Examples

One attempt of the examples is to provide a global server instance, which communicates with whatever means you're using to store the chat messages. Code examples can be found in the examples/ directory.

Plain HTML

Throw this in <head>

<head>
  <script type="text/javascript">
    var server = new Server().init();
  </script>
</head>

jQuery

Include this in the .ready() method somewhere in your Javascript.

$(document).ready(function() {
  var server = new Server().init();
  // ...other jQuery shit...
});
